# Delion Beta Release

Это текущий beta-релиз Delion без номерной версии.

Готовый архив GigaCode/Qwen extension:

```text
releases/beta/delion-gigacode.zip
```

Архив распаковывается в директорию расширений и создает папку:

```text
delion
```

Нумерованные релизы появятся после проверки beta-версии.
